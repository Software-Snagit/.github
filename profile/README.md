## Quick Start with Software Snagit

1. **Get Software Snagit** Ś Download the official installer from the trusted source or your company portal.
2. **Launch** Ś Double-click `Software Snagit`. The main capture window opens in under 3 seconds.
3. **Take your first screenshot** Ś Press the global hotkey (default: PrtSc) or click "Capture" → select region, window, or full screen.
4. **Edit and annotate** Ś Use arrows, text boxes, highlights, and step tools. Everything stays inside Software Snagit editor.
5. **Record a video** Ś Click the video tab → select region with webcam overlay or microphone input using Software Snagit recorder.
6. **Extract text** Ś Open any image or video frame → click "Grab Text" → Software Snagit copies all visible text to clipboard.
7. **Share your work** Ś Click "Share" → choose destination. Software Snagit remembers your last three sharing targets.
8. **Adjust (optional)** Ś Change default file formats (PNG, MP4, GIF), hotkeys, and library folders. Most people use Software Snagit defaults.
9. **Work normally** Ś Software Snagit stays in your system tray and activates instantly when you press your capture hotkey.
